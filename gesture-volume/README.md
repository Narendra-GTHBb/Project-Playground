# 🎧 Hand Volume Control

<p align="center">
  <b>Control your computer volume using hand gestures ✋</b><br>
  No keyboard. No mouse. Just your hand.
</p>

---

## ✨ Features

- ✋ Real-time hand tracking  
- 🔊 Volume control using thumb & index finger  
- ✊ Lock volume with middle finger  
- 📊 Smooth volume bar UI  
- ⚡ Fast & responsive  

---

## 🧠 How It Works

This project uses Computer Vision to detect your hand and control system volume:

- Detect hand using MediaPipe  
- Track finger positions  
- Calculate distance between fingers  
- Map distance → system volume  

---

## 🛠️ Tech Stack

- Python  
- OpenCV  
- MediaPipe  
- Pycaw
